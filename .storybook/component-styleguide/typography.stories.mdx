<h1 style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>Typography principles</h1>

<p style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>
  Text is the main way our users visualize content and complete their work.
</p>

- <p style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>
    Base font size and container widths are optimized for ideal line length: 50-75 characters
  </p>
- <p style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>
    Always provide sufficient color contrast with background: minimum 3.0 color ratio
  </p>

<br />

<h1 style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>Fonts</h1>

<p style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>
  Our default typography is Inter (Regular & Medium) and Libre Baskerville (Regular).
</p>

<br />

<h1 style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>Typography size scale</h1>

<p style={{ color: '#000000', fontFamily: 'Inter, sans-serif' }}>
  Our scale is based on a 1.125 increments, rounded up to pixel amounts.
  <br />
  Our base number is 16px for desktop and tablet, 14px for mobile.
</p>
